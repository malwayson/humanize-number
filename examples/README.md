# Examples

This folder contains examples demonstrating the features of `@malwayson/humanize-number`.

## Running Examples

First, build the project:

```bash
npm run build
```

Then run any example:

```bash
node dist/examples/comprehensive.js
node dist/examples/basic-usage.js
node dist/examples/advanced-features.js
node dist/examples/real-world.js
```

## Available Examples

### 1. `basic-usage.ts`

Introduction to core features and basic formatting.

### 2. `advanced-features.ts`

Demonstrates presets, localization, batch processing, and comparison utilities.

### 3. `real-world.ts`

Real-world use cases: dashboards, file uploads, system monitoring, e-commerce, fitness apps, weather apps.

### 4. `comprehensive.ts`

Complete showcase of all features in v2.0.

## Example Structure

Each example file is self-contained and can be run independently. They demonstrate:

- ✅ Basic number formatting
- ✅ All 10 format methods
- ✅ Unit system conversions
- ✅ Temperature, duration, speed, volume, percentage
- ✅ Presets and localization
- ✅ Batch processing
- ✅ Comparison utilities
- ✅ Error handling
- ✅ Real-world scenarios
